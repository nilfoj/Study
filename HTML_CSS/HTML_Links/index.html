Aqui está um resumo dos valores possíveis para o atributo target:

  1. _self: Abre o link na mesma janela ou aba que o link foi clicado. (Padrão)
  2. _blank: Abre o link em uma nova aba.
  3. _parent: Abre o link no frame pai.
  4. _top: Abre o link no corpo completo da janela.
  5. framename: Abre o link no frame especificado pelo nome.

O uso do rel

  • nofollow: indica que o link não deve ser seguido pelos mecanismos de busca.
  • follow: indica que o link deve ser seguido pelos mecanismos de busca.
  • sponsored: indica que o link é patrocinado.
  • ugc: indica que o link é gerado pelo usuário.
  • external: indica que o link é para um site externo.
