Modelo de Caixas

box-level

Ocupa todo espaço da linha

OBS: Um ótimo exemplo de uso é div e span! Ser o item ocupar toda a linha usa uma div, se for parte da linha usa span.

  • div
  • h1 - h6
  • p
  • main
  • header
  • nav
  • article
  • aside
  • footer
  • form
  • video

inline-level

Não ocupa a largura completo e os outros elementos seguem na mesma linha.

  • span
  • a
  • code
  • small
  • strong
  • em
  • sup e sub
  • label
  • button
  • input
  • select
